## Pengantar

Dokumentasi ini berisi informasi lengkap tentang semua komponen Vue yang digunakan dalam proyek Vyzor. Setiap komponen didokumentasikan dengan properti, slot, cara penggunaan, dan contoh kode.

---

## Struktur Folder

```
resources/
├── js/
│   ├── components/          # Komponen utama aplikasi
│   ├── shared/
│   │   └── @spk/           # Komponen shared yang dapat digunakan kembali
│   └── stores/             # Pinia stores untuk state management
└── UI/                     # Komponen UI tambahan
```

## Page Components

Komponen page adalah komponen-komponen halaman lengkap untuk berbagai fitur aplikasi.

### Applications

Komponen-komponen untuk berbagai aplikasi dan fitur produktivitas.

**Komponen:**
- Gallery - Halaman galeri dengan grid layout dan lightbox
- Full Calendar - Halaman kalender dengan drag & drop events
- File Manager - File manager dengan tree view dan file operations
- Chat - Aplikasi chat real-time dengan multiple conversations
- Task Management - Task management dengan kanban board dan list view
- Email Application - Email client dengan inbox, compose, dan settings
- To-Do List - Simple to-do list dengan categories dan priorities
- Sweet Alerts - Demo berbagai jenis alert dan notification

**Dokumentasi Lengkap:** [Applications Documentation](pages-components-documentation.md#applications-components)

---

### Dashboards

Komponen-komponen dashboard untuk berbagai domain bisnis dan industri.

**Kategori:**
- **General Dashboards:** Analytics, Sales, Social Media, Medical, Podcast, POS System, Stocks
- **E-commerce Dashboard:** Product management, Orders, Customers, Checkout
- **CRM Dashboard:** Leads, Deals, Contacts, Companies
- **Crypto Dashboard:** Portfolio, Wallet, Transactions, Market Data
- **Jobs Dashboard:** Job listings, Candidates, Companies, Applications
- **Projects Dashboard:** Project management, Overview, Creation
- **NFT Dashboard:** NFT portfolio, Marketplace, Auctions, Creation
- **Specialized:** HRM, School, Courses

**Dokumentasi Lengkap:** [Dashboards Documentation](pages-components-documentation.md#dashboard-components)

---

### Charts

Komponen-komponen untuk visualisasi data dengan berbagai library chart.

**Library:**
- **Chart.js:** Basic charts (line, bar, pie, doughnut)
- **Apex Charts:** Advanced charts (heatmap, funnel, candlestick, treemap)
- **ECharts:** 3D charts, Geographic data, Sankey diagrams

**Dokumentasi Lengkap:** [Charts Documentation](pages-components-documentation.md#charts-components)

---

### General UI

Komponen-komponen UI yang sering digunakan dalam aplikasi web.

- **Interactive Components:** Sortable, Swiper, Tours, Media Players, Draggable Cards

**Dokumentasi Lengkap:** [General UI Documentation](pages-components-documentation.md#general-ui-components)

## Panduan Penggunaan

### Menggunakan Komponen

Untuk menggunakan komponen dalam aplikasi Vue, ikuti langkah-langkah berikut:

1. Import komponen yang diperlukan:

```javascript
import ComponentName from '@/path/to/component.vue'
```

2. Daftarkan komponen dalam setup:

```javascript
<script setup>import ComponentName from '@/path/to/component.vue'</script>
```

3. Gunakan komponen dalam template:

```vue
<template>
  <ComponentName prop1="value1" prop2="value2">
    <!-- Slot content -->
  </ComponentName>
</template>
```

### Menggunakan Stores

Untuk menggunakan store dalam aplikasi Vue, ikuti langkah-langkah berikut:

1. Import store yang diperlukan:

```javascript
import { useStoreName } from '@/path/to/store.js'
```

2. Inisialisasi store dalam setup:

```javascript
<script setup>const store = useStoreName()</script>
```

3. Gunakan state dan actions dari store:

```vue
<template>
  <div>{{ store.stateProperty }}</div>
  <button @click="store.actionName()">Action</button>
</template>
```

### Kontribusi

Jika Anda ingin menambahkan atau memperbarui dokumentasi komponen:

1. Pastikan komponen telah didokumentasikan dengan benar
2. Tambahkan dokumentasi ke file yang sesuai
3. Perbarui file README.md dengan menambahkan komponen baru ke daftar isi
4. Pastikan semua link berfungsi dengan benar
